Comparable

Comparable只有一个方法compareTo(T)用来与其他对象比较,它的返回值有三种可能

  • 相等时返回0
  • 大于目标对象返回正数
  • 小于目标对象返回负数

如果目标对象无法与接收的对象比较的话,则会抛出ClassCastException.

Comparator

在List接口定义的方法中，也有一个排序方法sort(Comparator),这个方法并不需要元素对象本身实现Comparable接口，而是由我们来提供一个排序器Comparator，对元素进行比较排序。
