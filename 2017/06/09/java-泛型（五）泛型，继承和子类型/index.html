泛型类和子类型

以Collection类作为示例，ArrayList <E>实现List <E>，并且List <E>扩展Collection <E>。所以ArrayList <String>是List <String>的子类型，它是Collection <String>的子类型。只要不改变类型参数，子类型之间的关系将保留在类型之间。
